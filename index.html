Sobre este portafolio

Este sitio agrupa demos de sistemas desarrollados en PHP + MySQL + JS + CSS. Cada tarjeta abre una demo en una nueva pestaña para que puedas explorarlas en vivo. Los proyectos se entregan tal como fueron desarrollados; algunos requieren configuraciones propias (bases de datos, credenciales).
