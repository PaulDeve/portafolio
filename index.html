Sobre este portafolio

Este sitio muestra una colección de sistemas desarrollados en PHP + MySQL + JS + CSS. Si estás viendo esto en GitHub Pages (github.io), ten en cuenta que los demos no funcionarán aquí ya que requieren PHP y MySQL. Para ver los demos en acción:

  • 1. Descarga el código desde GitHub
  • 2. Instala XAMPP o usa Docker (instrucciones en el README)
  • 3. Ejecuta los demos localmente con soporte completo de PHP y MySQL
